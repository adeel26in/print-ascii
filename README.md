print-ascii

Render text in ASCII in the terminal

NOTE: It doesn't have a built-in help function just do: print-ascii [text]

Compile from source (You will have to figure your way in Windows! SORRY!):

Make sure you have Go installed: go.dev

git clone https://github.com/adeel26in/print-ascii.git

cd print-ascii/src

go build -o print-ascii

sudo mv print-ascii /usr/bin
